## 2021.12 月末

笔记系统升级。

当线性的知识笔记不能再满足我的需求的时候，是时候寻找一种网状笔记软件了。经过多维对比，最终选择了Obsidian。

但是我的知识结构正在从树形向网状进行转化，这就意味着线性的笔记不再适用

这也就意味着，这个repository的更新速度逐渐减慢，逐渐变成了字典式的查询工具，而不是idea的诞生地。
