## 结构

```c
#define MaxSize 50  
typedef struct{  
    int data[MaxSize];  
    int front, rear;  
}SqQueue;
```

## 基本操作

- `InitQueue(&Q)`：初始化队列, 构造一个空队列Q。
- `QueueEmpty(Q)`：判队列空, 若队列Q为空返回true, 否则返回false。
- `EnQueue(&Q, x)`：入队, 若队列Q未满, 将x加入, 使之成为新的队尾。
- `DeQueue(&Q, &x)`：出队, 若队列Q非空, 删除队头元素, 并用x返回。
- `GetHead(Q, &x)`：读队头元素, 若队列Q非空, 则将队头元素赋值给x。
