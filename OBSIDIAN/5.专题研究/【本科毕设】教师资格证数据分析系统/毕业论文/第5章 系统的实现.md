# 第5章 系统的实现

## 登录功能实现

用户想要进入系统进行相关操作，必须先进行登录。如图所示，显示的是本系统的登录界面，页面由标题、两个输入框和登录按钮组成，用户需要输入用户名和密码，然后点击登录按钮进行登录。

![[assets/Pasted image 20230426152920.png]]

用户登录成功之后会成功登录进入系统。如下图所示，页面的左侧为导航栏，右侧为页面主体。

![[assets/Pasted image 20230427184952.png]]

如果用户登录失败，也会在页面上方显示“登录失败”，此时用户需要重新登录。

![[assets/Pasted image 20230427181135.png]]

## 数据大屏功能实现

### 数据看板

进入系统之后显示的是数据看板界面，如图所示，数据看板上方是时间信息，下方是数据总数列表和数据变化趋势：数据总数列表上方是到达数据大屏的链接，其他分别是认定信息、笔试成绩、面试成绩和证书信息的总数，并且在右边有四种信息按照时间变化的趋势。

![[assets/Pasted image 20230427184952.png]]

将鼠标放到特定的时间上面的时候，会立即显示出来当前年份的人数总数，如图所示。

![[assets/Pasted image 20230427185011.png]]

### 数据大屏

点击数据看板页面的查看数据大屏的按钮，会跳转到如图所示的数据大屏页面。

数据大屏上会显示如下信息：当年最高学位、最高学历的柱状图，性别分布、专业类别、考试类型，面试等级的饼图，以及在右下角显示任教学科的词云图。

![[assets/Pasted image 20230426152945.png]]

## 数据管理功能实现

### 笔试成绩管理

笔试成绩管理页面如下所示，主要由两大部分组成， 上方是搜索框，下方是表格，为保护考生隐私，下图对部分数据做了马赛克处理。

![[assets/Pasted image 20230427202758.png]]

笔试成绩管理由笔试成绩查询、添加、删除、修改，导入和导出组成。以下是详细实现：

（1）笔试成绩查询

成绩查询界面的上方是搜索界面，用户可以通过填写相关的搜索字段并点击相关的按钮进行搜索，同时也可以筛选教育学、教育心理学、职业道德修养和高等教育法规的考试状态和按照三个学科的考试成绩进行排序。

如图所示，选择了所有教育学考试正常的，女生成绩降序排序，查询出来的数据如图所示

![[assets/Pasted image 20230427203003.png]]

（2）笔试成绩添加

如图，点击添加按钮之后，弹出了添加考生信息的对话框，如图所示，用户填写完添加的表单之后点击添加即可直接添加考试信息。

![[assets/Pasted image 20230427203746.png]]

其中，表单中的教育学考试状态、教育心理学考试状态、职业道德修养和高等教育法规状态三者，如果某一科考试状态不是正常的时候，将无法填写这个科目的成绩。如图所示，选择了非正常的考试状态的时候，无法填写相关的成绩。

![[assets/Pasted image 20230427203634.png]]

（3）笔试成绩修改

点击表格列表右侧的编辑按钮，可以对特定人的信息进行修改，点击特定信息行右侧修改按钮之后，页面中会弹出修改考生信息的对话框，如图所示，修改当前考生信息之后点击修改按钮直接提交修改即可。

![[assets/Pasted image 20230427204116.png]]

（4）笔试成绩删除

点击表格列表右侧的删除按钮可以删除特定信息，为了防止用户误删信息，在用户弹出删除按钮之后会让用户确认是否删除，在用户点击确定之后才删除信息。

![[assets/Pasted image 20230427204546.png]]

（5）笔试成绩导出

笔试成绩导出有两种方式，可以通过表格左侧的筛选方框进行数据导出，也可以通过搜索进行导出。在导出的时候还可以选择导出列，如图所示，在导出的时候可以自行选择导出列。选择完毕之后直接点击导出即可。

![[assets/Pasted image 20230427204817.png]]

如图所示，导出了第一列选中数据到Excel表格中。

![[assets/Pasted image 20230427213934.png]]

（6）笔试成绩导入

用户也可以从Excel表格导入成绩，点击表格上方，右侧的导入按钮之后，用户可以选择自己电脑上的Excel文件进行导入。如图所示，点击导入文件即可。

![[assets/Pasted image 20230427205046.png]]

## 用户管理功能实现

用户管理是本系统的重要功能之一，用户管理的主要功能是对用户进行管理，主要操作包括添加用户、修改密码和删除用户，因为使用本系统的人数不太多，因此没有搜索用户功能。用户管理的界面如图所示。

![[assets/Pasted image 20230427201007.png]]

### 添加用户

点击页面右上方的添加用户按钮可以添加用户，在对话框中填写相关的信息之后点击提交可以添加用户。

![[assets/Pasted image 20230427201015.png]]

### 修改密码

点击表格列表右边的修改密码按钮可以修改用户的密码，如图所示，修改密码的时候需要填写旧密码和两次新密码，填写完成之后点击提交可以修改用户密码。

![[assets/Pasted image 20230427201021.png]]

### 删除用户

表格列表右边的操作栏中有删除用户的按钮，点击删除按钮会弹出确认的气泡，如图所示，为了防止用户删除不想删除的数据，需要用户进行二次确认之后才能进行删除。

![[assets/Pasted image 20230427201337.png]]
