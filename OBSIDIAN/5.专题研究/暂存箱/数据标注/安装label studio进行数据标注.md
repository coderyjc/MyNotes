### 1. 安装与启动

采用Anaconda创建名为label_sudio的虚拟环境

```bash
conda create -n label_studio python=3.9
```

 激活该虚拟环境

```bash
conda activate label_studio
```

 安装Label Studio

```bash
pip install label-studio
```

启动Label Studio

```bash
label-studio start
```
