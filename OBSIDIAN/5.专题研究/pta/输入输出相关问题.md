## 结尾是特殊字符
看清楚是以负整数结尾还是-1结尾。

## 读取一行数据

### C++

直接使用`cin`读取，如果一行结束，会返回null

```cpp
int a;
while(cin >> a){
	// do something
}
```

---

使用`stringstream`读取一行，然后依次输入

```cpp
#include<sstream>

string line;
getline(cin, line);
ss.str(line);

int a;

ss >> a;
```


### C

scanf之后，使用getchar读取末尾字符，如果是`\n`，就是一行输入结束

```cpp
char c;
while(c != '\n'){
	scanf("...");
	c = getchar();
	
// do something
}
```
